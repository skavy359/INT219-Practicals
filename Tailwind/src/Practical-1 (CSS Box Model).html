Tailwind CSS Box Model Example

Box Model

This box demonstrates the properties of the box model using Tailwind CSS.

Padding

This box has p-4, which means padding of 1rem(16px).

Border

This box has border-4, a dashed border with border-blue-300.

Margin

This box is separated from others using mb-4 which is margin-bottom.

Width and Height

This box has w-64 (16-rem) h-32 (8-rem).
